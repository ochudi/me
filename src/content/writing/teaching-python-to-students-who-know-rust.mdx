Placeholder introduction. A handful of my students learned Rust first.
Watching them meet Python is instructive in both directions.

In Rust they are used to the compiler holding the invariants:

```rust
struct Student {
    name: String,
    score: u32,
}

impl Student {
    fn passed(&self) -> bool {
        self.score >= 50
    }
}
```

Python will happily accept the same shape and none of the guarantees:

```python
class Student:
    def __init__(self, name: str, score: int) -> None:
        self.name = name
        self.score = score

    def passed(self) -> bool:
        return self.score >= 50
```
